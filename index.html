Tasy EMR • Gestão Hospitalar

🏥 Extração de Biblioteca de Fármacos para Bombas de Infusão

1. O que é o projeto

Desenvolvimento de uma consulta SQL avançada para a extração de dados técnicos de medicamentos do ERP Tasy para bombas B Braun no HMG.

2. O Problema Técnico

  • Dados espalhados em mais de 9 tabelas diferentes.
  • Hierarquia inconsistente entre capa do material e prescrição.

3. A Solução Aplicada

Técnicas avançadas de SQL: Joins Complexos, NVL, NULLIF e DECODE/CASE.

4. O Impacto na Ponta

Segurança do Paciente e redução drástica de retrabalho manual.

SQL Avançado • Suprimentos

🚀 Otimização: Relatório de Consumo Diário por Estoque

1. O que é o projeto?

Reestruturação do relatório WATE-1570 no Tasy EMR para parametrização de cotas.

2. O Problema Técnico

  • Erro de Join multiplicando valores; documentos de 74 páginas inúteis.

3. A Solução Aplicada

Refatoração do SQL com HAVING SUM(qt_consumo) > 0 e filtragem por unidade.

4. O Impacto na Ponta

Redução para 8 páginas úteis e precisão na reposição de insumos.
