Tasy EMR • SQL

🏥 Extração de Biblioteca de Fármacos para Bombas de Infusão

Extração de dados técnicos (doses, diluições e estabilidades) do ERP Tasy para configuração das bombas de infusão B Braun na unidade hospitalar HMG.

  • Problema: Dados dispersos em +9 tabelas e siglas internas ilegíveis.
  • Solução: Consulta SQL com múltiplos LEFT JOINs, NVL/NULLIF e normalização de códigos.
  • Impacto: Segurança do paciente e economia de horas de trabalho manual.

🚀 Customização de Regras de Negócio: Controle de Requisição

Implementação de trava lógica no módulo de Requisição de Materiais para garantir a rastreabilidade financeira.

  • Problema: Materiais consumidos não eram faturados por falta de vínculo com o atendimento.
  • Solução: Regra de Atributo via SQL condicionando a obrigatoriedade do campo Atendimento.
  • Impacto: Garantia de receita e integridade total dos dados de inventário.
