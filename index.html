Tasy EMR • Gestão Hospitalar

🏥 Extração de Biblioteca de Fármacos para Bombas de Infusão

1. O que é o projeto

Desenvolvimento de uma consulta SQL avançada para a extração de dados técnicos de medicamentos do ERP Tasy para as bombas B Braun no HMG.

2. O Problema Técnico

  • Dados dispersos em mais de 9 tabelas.
  • Hierarquia de dados inconsistente entre "capa" e "prescrição".
  • Variações de via não vinculadas a regras de estabilidade.

3. A Solução Aplicada

Joins complexos, lógica de Fallback com NVL e normalização com DECODE e CASE.

4. O Impacto na Ponta

Segurança do paciente com configuração exata e redução drástica de retrabalho manual.

🛠️ Tecnologias: Oracle SQL, Tasy EMR.
SQL Avançado • Suprimentos

🚀 Otimização: Relatório de Consumo Diário por Estoque

1. O que é o projeto

Reestruturação do relatório WATE-1570 no Tasy para parametrização de cotas.

2. O Problema Técnico

  • Produto cartesiano multiplicando valores indevidamente.
  • Documentos de 74 páginas com 90% de ruído.

3. A Solução Aplicada

Refatoração de joins e aplicação de HAVING SUM(qt_consumo) > 0.

4. O Impacto na Ponta

Redução para 8 páginas de dados úteis e precisão na reposição de medicamentos.
