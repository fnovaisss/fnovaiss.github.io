Tasy EMR • Gestão Hospitalar

🏥 Extração de Biblioteca de Fármacos para Bombas de Infusão

1. O que é o projeto

Desenvolvimento de uma consulta SQL avançada para a extração de dados técnicos de medicamentos do ERP Tasy para as bombas B Braun no HMG.

2. O Problema Técnico

  • Dados Dispersos em 9+ tabelas.
  • Hierarquia inconsistente entre capa do material e aba de prescrição.

3. A Solução Aplicada

Joins Complexos, Lógica de Fallback com NVL e normalização via DECODE/CASE.

4. O Impacto na Ponta

Segurança do Paciente e redução drástica de retrabalho manual.

🛠️ Tecnologias: Oracle SQL, Tasy EMR.
SQL Avançado • Suprimentos

🚀 Otimização: Relatório de Consumo Diário por Estoque

1. O que é o projeto?

Reestruturação do relatório WATE-1570 no Tasy EMR para parametrização de cotas.

2. O Problema Técnico

  • Produto cartesiano multiplicando valores indevidamente.
  • Documentos de 74 páginas com 90% de ruído (consumo zero).

3. A Solução Aplicada

Refatoração do SQL corrigindo ligações e aplicação de HAVING SUM(qt_consumo) > 0.

4. O Impacto na Ponta

Redução para 8 páginas úteis e precisão na reposição de medicamentos.
